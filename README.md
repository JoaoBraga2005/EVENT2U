# EVENT2U
Trabalho da matéria Análise de Projeto de Software Orientado a Objetos, sobre a aplicação hibrída (com elementos desktop e web) EVENT2U (Sistema de eventos).
Desenvolvido por: João Vitor Costa Braga, Louise Mayumi Takigawa Pereira, Maria Eduarda Justino e Mariana Paulino.


Como rodar a aplicação:

1. Baixe os requirements:
(no terminal escreva)
--> pip install -r requirements.txt

2. Rode a aplicação: 
--> python Projeto/run.py

3. Aplicação aberta!
